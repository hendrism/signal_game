# SIGNAL — 2.0 Ideas (parked, do not build before 1.0)
This is the file §12 of the design doc tells agents to write into instead of adding features. Anything here is *approved as an idea only*.

## Parked with intent (Fable-reviewed, likely 2.0 headliners)
- **Weather layer.** Zones cycle simple conditions on a slow clock (dust season: −50% Ashfields yields, ×2 anomaly odds). Adds "when to explore," not just "where." Parked because it touches balance everywhere; design it as its own pass, not a patch.
- **Ghost routes.** Rare finds start a 3-step mini-expedition chain telling one complete small story (e.g., Okoye's last inspection round) with a unique reward. Parked because expedition *chains* are a new engine mechanic.
- **New game plus.** Post-Beacon restart on a neighboring planet: remixed discovery tables, one carried-over hub memory referencing the player's ending choice. The right way to make finishing feel like an invitation.
- **Shareable network codes.** Compress a save into a string others can *view* as a constellation (not play). Show-your-factory energy, single-player safe.
